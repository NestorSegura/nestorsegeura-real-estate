  <action>
    **Navbar.tsx (Server Component):**
    - Import `sanityFetch` from `@/sanity/lib/live` and `SITE_SETTINGS_QUERY` from `@/sanity/lib/queries`
    - Fetch siteSettings, extract `navigation[]` (array of {label, href}) and `defaultCtaHref`
    - Pass navLinks, ctaHref, and siteName as props to `NavbarClient`
    - Provide sensible fallbacks: empty array for navLinks, `'#kontakt'` for ctaHref

    **NavbarClient.tsx ('use client'):**
    - Receives props: `navLinks: {label: string; href: string; _key: string}[]`, `ctaHref: string`, `siteName: string`
    - **Sticky scroll detection:** Use IntersectionObserver on a sentinel ref. The sentinel `<div>` is rendered inside NavbarClient at the very top (before the nav element), with `className="h-0 w-full"` and `aria-hidden="true"`. When sentinel is not intersecting, set `scrolled=true`.
    - **Nav styling:** `position: sticky; top: 0; z-index: 50`. When `scrolled`: add `backdrop-filter: blur(12px)`, `background: oklch(0.12 0 0 / 0.85)` (dark translucent), `box-shadow`. Transition all properties with `duration-300`.
    - **Desktop layout (md+):** Logo/siteName on left, nav links center, language switcher + CTA button on right.
    - **Nav links:** Render each navLink as an `<a>` with `href={link.href}` (these are anchor links like `#leistungen`). Style: text-sm font-medium, hover color `oklch(0.72 0.14 290)`.
    - **CTA button:** "Erstgespräch vereinbaren" text (from `nav.cta` translation), links to `ctaHref`, opens in new tab (`target="_blank" rel="noopener noreferrer"`). Style: filled primary `oklch(0.45 0.18 290)` background, white text, rounded-full, px-6 py-2.5.
    - **Language switcher:** Render `DE | EN | ES` text labels. Use `useLocale()` from `next-intl`, `usePathname()` and `useRouter()` from `@/i18n/navigation`, `useParams()` from `next/navigation`. Active locale gets `oklch(0.72 0.14 290)` color and font-bold. Click calls `router.replace({ pathname, params }, { locale: code })`.
    - **Mobile (below md):** Hide nav links and show hamburger button (Menu icon from lucide-react). Use `@base-ui/react` Drawer component:
      - `<Drawer.Root>` wrapping `<Drawer.Trigger>` (hamburger button)
      - `<Drawer.Portal>` containing `<Drawer.Backdrop>` (fixed inset-0 bg-black/40) and `<Drawer.Viewport>` (fixed inset-y-0 right-0)
      - `<Drawer.Popup>` styled as `h-full w-72 bg-white flex flex-col p-6` with close button (X icon) via `<Drawer.Close>`
      - Inside popup: nav links as vertical list, language switcher, CTA button
      - Drawer slides in from the right
    - **Important:** All navigation links use plain `<a>` tags for anchor links (same-page scroll). The CTA link also uses `<a>` since it's an external URL. Do NOT use `@/i18n/navigation` Link for anchor or external links -- Link is for internal page navigation only. The locale switcher IS the only place that uses `useRouter` from `@/i18n/navigation`.

    **Layout update (src/app/[locale]/layout.tsx):**
    - Import `Navbar` from `@/components/Navbar`
    - Add `<Navbar />` above `<main>{children}</main>` wrapper
    - Wrap `{children}` in a `<main>` tag if not already wrapped (check current structure)
    - Keep `<SanityLive />` after main

    **CSS (src/app/globals.css):**
    - Add rule: `section[id] { scroll-margin-top: 80px; }` to account for sticky navbar height when anchor-scrolling

    **Message files (de.json, en.json, es.json):**
    - Add/update `nav.cta` key:
      - DE: "Erstgespräch vereinbaren"
      - EN: "Book Consultation"
      - ES: "Agendar consulta"
    - Add `nav.menuOpen` and `nav.menuClose` for aria-labels:
      - DE: "Menü öffnen" / "Menü schließen"
      - EN: "Open menu" / "Close menu"
      - ES: "Abrir menú" / "Cerrar menú"
  </action>
