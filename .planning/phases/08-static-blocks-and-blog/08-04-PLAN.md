  <action>
Create a single SEO helper module exporting:

`buildMeta({ pageSeo, postSeo, settings, locale, fallbackTitle, fallbackDescription, ogImageRef, currentUrl })` → returns `{ title, description, ogTitle, ogDescription, ogImage, ogType, canonical }`. Implements the resolution chain from CONTEXT.md:
- title: pageSeo.title → settings.seo.title → fallbackTitle
- description: pageSeo.description → settings.seo.description → fallbackDescription
- ogImage: ogImageRef (page's seo.ogImage, resolved via urlFor) → settings.seo.ogImage (resolved via urlFor) → `${site}/og-default.png`
- canonical: currentUrl (Astro.url.href)

`buildOrganizationSchema(settings, siteUrl)` → `{ '@context':'https://schema.org', '@type':'Organization', name, url, logo? }`.

`buildPersonSchema(settings)` → `{ '@context':'https://schema.org', '@type':'Person', name, jobTitle?, image?, sameAs:[] }` from siteSettings (use siteName as name, tagline as jobTitle, ogImage as image, footer.socialLinks as sameAs).

`buildArticleSchema({ post, locale, siteUrl, postUrl })` → `{ '@type':'Article', headline:post.title, datePublished, dateModified?, image:resolvedMainImageUrl, author:{ '@type':'Person', name:post.author.name }, description:post.excerpt }`.

`buildBreadcrumbSchema(items: Array<{name, url}>)` → `{ '@type':'BreadcrumbList', itemListElement: items.map((it,i) => ({ '@type':'ListItem', position:i+1, name:it.name, item:it.url })) }`.

Use `urlFor()` from imageUrl.ts to resolve image refs. Read site URL from `import.meta.env.PUBLIC_SITE_URL ?? 'https://realestatestrategy.eu'`.
