     - `sitemap: 'https://nestorsegura.com/sitemap.xml'`

2. Create `src/app/sitemap.ts`:
   - Import `MetadataRoute` from `next`
   - Import `getPathname` from `@/i18n/navigation`
   - Import `sanityFetch` from `@/sanity/lib/live`
   - Import `ALL_POSTS_FOR_SITEMAP_QUERY` from `@/sanity/lib/queries`
   - Define `BASE_URL = 'https://nestorsegura.com'`
   - Define `LOCALES = ['de', 'en', 'es'] as const`
   - Create helper `buildAlternates(href: string)`:
     - For each locale, compute URL: `BASE_URL + getPathname({ locale, href })`
     - Return `{ languages: { de: url_de, en: url_en, es: url_es } }`
   - Export default async function returning `MetadataRoute.Sitemap`:
     - Static pages (each locale variant as a separate entry with alternates):
       - Homepage: priority 1.0, changeFrequency 'monthly', alternates via buildAlternates('/')
       - /analyse: priority 0.9, changeFrequency 'monthly', alternates via buildAlternates('/analyse')
       - /blog: priority 0.8, changeFrequency 'weekly', alternates via buildAlternates('/blog')
     - For static pages, generate one entry PER locale (German URL at /, English at /en, Spanish at /es, etc.) each with the same alternates object
     - Dynamic blog posts: fetch `ALL_POSTS_FOR_SITEMAP_QUERY`, map each post to an entry:
       - `url: BASE_URL + getPathname({ locale: post.language ?? 'de', href: '/blog/' + post.slug.current })`
       - `lastModified: post._updatedAt ? new Date(post._updatedAt) : new Date()`
       - `changeFrequency: 'weekly'`
       - `priority: 0.7`
       - `alternates: buildAlternates('/blog/' + post.slug.current)`
     - Return combined array

   CRITICAL: German (default locale) pages have NO prefix. `getPathname({ locale: 'de', href: '/blog' })` returns `/blog`, not `/de/blog`. English returns `/en/blog`, Spanish `/es/blog`. Do NOT hardcode locale prefixes.
