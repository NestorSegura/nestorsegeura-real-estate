4. **Cookie-based locale persistence script** — only in `src/pages/index.astro` (the DE root). Add inside BaseLayout children (or as a `<script is:inline>` tag):

```html
<script is:inline>
  (function() {
    var m = document.cookie.match(/(?:^|; )ns_locale=([^;]+)/);
    var loc = m && m[1];
    if (loc && loc !== 'de' && !sessionStorage.getItem('ns_locale_redirected')) {
      var map = { en: '/en', es: '/es' };
      if (map[loc]) {
        sessionStorage.setItem('ns_locale_redirected', '1');
        window.location.replace(map[loc]);
      }
    }
  })();
</script>
```

The script is **only** on `/` (DEFAULT locale root). Deeper URLs always win per locked decision. Do NOT add this script to any other page.
